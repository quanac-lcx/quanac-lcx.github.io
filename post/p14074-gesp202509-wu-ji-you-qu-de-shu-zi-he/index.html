P14074 [GESP202509 五级] 有趣的数字和

2025-10-26 2 min read

https://www.luogu.com.cn/problem/P14074

这道题目，暴力只能拿到40分。

但是仔细观察114514年就可以发现，4个二进制数为一个周期循环，奇数1和偶数1的个数一致。

比如0-3 二进制是：0 1 10 11，偶数个一的数有：0 11，奇数：1 10，奇数个一的和就是0+3=3，偶数为1+2=3，3=3.

然后可以发现公式：

总和=区间个数2∗4−区间个数总和=区间个数^2*4-区间个数 总和=区间个数2∗4−区间个数

然后写代码：

#include<bits/stdc++.h>
using namespace std;
typedef unsigned long long ll;

ll arr[]={0,1,3,3,7,7,7,14,22};

bool check(int n){
	int sum1=0;
	//转化为二进制
	string a=bitset<64>(n).to_string();
	for(int i=0;i<a.size();i++){
		if(a[i]=='1')
			sum1++;
	}
	return sum1%2==1;
}
ll f(ll n){
	if(n<=8) return arr[n]; //防止减出负数
	ll shuLiang=(n-4)/4;
	ll ans=shuLiang*shuLiang*4-shuLiang;
	for(int i=shuLiang*4;i<=n;i++){
		check(i)?ans+=i:0;
	}
	return ans;
}
int main(){
	ll l,r,ans=0;
	cin>>l>>r;
	cout<<f(r)-f(l-1);
	return 0;
}
